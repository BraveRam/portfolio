## backblaze-mcp

- [BraveRam/backblaze-mcp](https://github.com/BraveRam/backblaze-mcp)
  - A professional Model Context Protocol (MCP) server implementation providing seamless integration with Backblaze B2 cloud storage services. Offers a comprehensive suite of tools for bucket lifecycle management, advanced file operations (including large file multipart uploads), API key administration, and access control. Architected as an intelligent wrapper around the backblaze-b2 npm package, adding MCP-compliant interfaces for AI assistants and automated workflows. Enables efficient cloud storage operations through standardized protocol communication.
